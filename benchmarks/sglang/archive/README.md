# SGLang Benchmark Archive

This folder preserves failed or superseded SGLang benchmark-readiness attempts.
It is kept for audit and debugging history, not as the current benchmark result
surface.

Use [`../README.md`](../README.md) for the SGLang benchmark status and
[`../../current/README.md`](../../current/README.md) for the concise current
benchmark summary.

Archived folders should keep compact sanitized records only. Raw run payloads,
logs, credentials, wheels, and generated datasets stay out of Git.
