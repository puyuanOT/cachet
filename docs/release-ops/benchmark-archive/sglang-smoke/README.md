# SGLang Smoke Archive

This folder preserves failed, superseded, and readiness-only SGLang live
handoff attempts. It is maintained for debugging and release history, not as
the current public benchmark result surface.

Use [`../../../../benchmarks/sglang/README.md`](../../../../benchmarks/sglang/README.md)
for current SGLang benchmark status and
[`../../../../benchmarks/current/README.md`](../../../../benchmarks/current/README.md)
for the concise performance summary.

Archived folders keep compact sanitized records only. Raw run payloads, logs,
credentials, wheels, generated prompts, and local scratch output stay out of
Git.
