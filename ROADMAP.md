# Roadmap

Cachet is alpha software. The near-term roadmap is:

- polish the stable `cachet` Python API around local workflows and handoff
  bundles;
- provide clearer examples for real Transformers-backed KV generation;
- continue hardening vLLM native KV-transfer integration;
- continue validating SGLang HiCache handoff behavior on realistic prompts;
- consolidate user-facing CLIs behind a smaller `cachet` command group;
- move bulky audit artifacts to release assets or clearly labeled archives.

Good first issues should be small documentation, examples, validation, or test
coverage tasks that do not require Databricks or GPU access.
